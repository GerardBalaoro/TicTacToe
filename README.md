### Clone / Fork Guide

1. Install virtualenv library
   
   ```
   pip install virtualenv
   ```

2. Setup virtual environment and activate
   
   ```
   python3 -m virtualenv env
   ```

   - **Windows:**

		```
		./env/bin/activate
		```

   - **Linux/Unix:** 

		```
		source env/bin/activate
		```

3. Install requirements
   
   ```
   pip install requirements.txt
   ```
